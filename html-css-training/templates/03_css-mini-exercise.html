✨ ミッション1: h1の色を変える（5分）

目標: 一番上の「CSSミニワーク」の文字色を好きな色に変えましょう

方法: 上の <style> タグの中に、以下のように書いてください

h1 {
  color: blue;  /* ← 好きな色に変える */
}

使える色の例: red, blue, green, purple, orange, pink, brown, navy

💡 わからなければAIに聞こう: 「CSSでh1の色を変える方法は？」

✅ 完成の目安: ページ上部の「CSSミニワーク」の色が変わっていればOK！

📦 ミッション2: 箱に余白をつける（5分）

目標: この灰色の箱に、内側の余白（padding）と外側の余白（margin）をつけましょう

方法: 上の <style> タグの中の .box に、以下を追加してください

.box {
  border: 2px solid #333;
  background-color: #f0f0f0;
  padding: 20px;   /* ← この行を追加 */
  margin: 15px;    /* ← この行を追加 */
}

違いを確認:

  • padding: 箱の内側の余白（文字と枠の間）
  • margin: 箱の外側の余白（箱と箱の間）

💡 数字を変えて試してみよう: padding: 50px; にすると余白が広くなる！

✅ 完成の目安: 箱の文字と枠の間に余白ができて、箱同士の間隔が空いていればOK！

🔍 ミッション3: Chrome検証ツールで色をコピーする（5分）

目標: 他のサイトで使われている色を、Chrome検証ツールでコピーして、この箱に適用しましょう

手順:

  1. 新しいタブでGoogleやYahoo!などのサイトを開く
  2. F12キーを押す（Macの場合: Cmd + Option + I）
  3. 左上の矢印アイコン（要素選択）をクリック
  4. 気に入った色の部分（ボタンやリンクなど）をクリック
  5. 右側のStylesパネルで color や background-color の値を確認
  6. 値をコピーして、上の <style> タグの .mission3 に貼り付ける
.mission3 {
  color: #1a73e8;           /* ← コピーした色を貼り付け */
  background-color: #f8f9fa; /* ← コピーした色を貼り付け */
}

💡 色の書き方はいろいろ:

  • 色の名前: red, blue
  • 16進数: #1a73e8, #ff0000
  • RGB: rgb(26, 115, 232)

✅ 完成の目安: この箱の文字色や背景色が変わっていればOK！
