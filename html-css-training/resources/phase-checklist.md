# Phase別チェックリスト（講師用）

このチェックリストは、講師がブレイクアウトルームを巡回する際に、各チームの進捗を確認するために使用します。

---

## Phase 1（0-25分）: 土台作り

**目標**: navbar + ヒーローセクション完成

### チェック項目

- [ ] **navbar が表示されているか**
  - サイト名が設定されているか
  - 背景色が適用されているか

- [ ] **ヒーローセクション（h1+p）があるか**
  - h1（大見出し）が設定されているか
  - p.lead（サブタイトル）が設定されているか
  - 中央揃えになっているか

- [ ] **Live Serverで表示確認できているか**
  - ブラウザでページが開けているか
  - 変更が反映されているか（自動リロード確認）

### よくある問題と対処法

- **Live Serverが起動しない** → VSCodeでHTMLファイルを右クリック → "Open with Live Server"
- **navbarが表示されない** → Bootstrapのリンクが正しいか確認
- **中央揃えにならない** → text-centerクラスが付いているか確認

---

## Phase 2（25-55分）: コンテンツ充実

**目標**: card 3つ + 説明文100字以上完成

### チェック項目

- [ ] **card 3つ表示されているか**
  - 3つのカードが横に並んでいるか（PC表示時）
  - card-title、card-text、btnが設定されているか

- [ ] **画像3枚使用されているか**
  - 各カードに画像が設定されているか
  - 画像が表示されているか（リンク切れでないか）
  - できればプレースホルダーから変更されているか

- [ ] **説明文100字以上あるか**
  - 「詳しい説明」セクションに文章があるか
  - 100文字以上書かれているか（目視確認でOK）

- [ ] **レスポンシブ対応（col-md-*）使用しているか**
  - col-md-4 などのグリッドクラスが使われているか
  - ブラウザ幅を狭めるとカードが縦に並ぶか

### よくある問題と対処法

- **カードが縦に並んでしまう** → col-md-4 クラスが付いているか確認
- **画像が表示されない** → URLが正しいか、インターネット接続があるか確認
- **カードの高さがバラバラ** → h-100 クラスを追加することを提案

---

## Phase 3（55-85分）: 磨き上げ

**目標**: カスタマイズ + 最終調整

### チェック項目

- [ ] **カスタマイズ（色、画像、レイアウト）されているか**
  - navbar の色が変更されているか（navbar-dark bg-primary など）
  - ボタンの色が変更されているか（btn-success など）
  - 背景色やテキスト色が追加されているか

- [ ] **画像が適切に選ばれているか**
  - テーマに合った画像が使われているか
  - Unsplashなどから取得した高品質な画像か

- [ ] **推奨要素が追加されているか**（オプション）
  - フッター
  - Badge（ラベル）
  - shadow（影）
  - 追加のセクション

- [ ] **全体的な見た目が整っているか**
  - 余白（margin/padding）が適切か
  - 文章にtypoがないか
  - リンクが動作するか

### よくある問題と対処法

- **色の変更方法がわからない** → navbar-light を navbar-dark に、bg-light を bg-primary に変更
- **画像の探し方がわからない** → Unsplash.com でキーワード検索
- **時間が余っている** → Badge、shadow、追加セクション、アニメーションなどを提案

---

## 全体的なチェックポイント

### 25分時点（Phase 1 終了）

- **全員メインルームに戻す**
- 「Phase 1完了した人？」と挙手確認
- 未完成のチームには残り時間の使い方をアドバイス
- 完成したチームを称賛

### 55分時点（Phase 2 終了）

- **全員メインルームに戻す**
- 「Phase 2完了した人？」と挙手確認
- ここまでで必須要件は達成されていることを確認
- Phase 3は「磨き上げ」であることを強調

### 70分時点（Phase 3 途中）

- 「あと15分！仕上げに入りましょう」とアナウンス
- 未完成のチームには優先順位を伝える

### 80分時点（Phase 3 終了5分前）

- 「あと5分！提出準備を」とアナウンス
- 最終確認を促す

## トラブルシューティング

### よくあるエラー

| エラー内容 | 原因 | 解決方法 |
|-----------|------|---------|
| Live Serverが起動しない | 拡張機能が入っていない | VSCode拡張機能からLive Serverをインストール |
| Bootstrapが効いていない | CDNリンクが間違っている | テンプレートからコピペし直す |
| カードが縦に並ぶ | col-md-4 が抜けている | div に class="col-md-4" を追加 |
| 画像が表示されない | URLが間違っている | コピペミスがないか確認 |
| 文字化け | 文字コードが間違っている | <meta charset="UTF-8"> があるか確認 |
