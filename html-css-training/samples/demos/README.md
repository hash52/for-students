# デモサンプル一覧

このフォルダには、研修で使用する解説用のデモファイルが含まれています。

## 📁 ファイル一覧

### HTML基礎デモ

**demo-01-html-semantic.html** - セマンティックHTML
HTMLは「文章構造」をマークアップするものであることを示すデモ。見出し、段落、リストなどの意味的な役割を学びます。

**demo-02-html-nesting.html** - タグの入れ子構造
HTMLの階層構造（箱の中に箱を入れる）を視覚的に理解するデモ。色分けで構造を分かりやすく表示。

### CSSデモ

**demo-03-css-box-model.html** - CSSボックスモデル
padding、margin、borderの違いを実際に見て理解するデモ。Chrome検証ツールでの確認方法も説明。

**demo-04-css-colors.html** - CSSの色指定
文字色（color）、背景色（background-color）、ボーダーの色など、色の指定方法を学ぶデモ。

### Bootstrapグリッドデモ

**demo-05-grid-2column.html** - Bootstrap 2カラムグリッド
2カラムレイアウトの作り方と、レスポンシブ対応を学ぶデモ。col-md-6、col-md-8/4などの使い方。

**demo-06-grid-3column.html** - Bootstrap 3カラムグリッド
3カラムレイアウト（col-md-4）の実用例。商品一覧、サービス紹介などでよく使うパターン。

### Bootstrapコンポーネントデモ

**demo-07-navbar.html** - Bootstrapナビゲーションバー
navbarの基本から、色の変更、ボタン追加、レスポンシブメニューまで。コピペ用テンプレート付き。

**demo-08-card.html** - Bootstrapカード
カードコンポーネントの基本から、画像付きカード、3カラムカード、横並びカードまで。実用例多数。

**demo-09-responsive.html** - レスポンシブデザイン
画面サイズに応じてレイアウトが変わる仕組みを学ぶデモ。col-lg-*, col-md-*, col-sm-*の使い分け。

### Bootstrap導入前後の比較

**demo-10-before-bootstrap.html** - Bootstrap導入前
Bootstrapを使わずに全てのCSSを自分で書いた場合の例。大変さを実感。

**demo-11-after-bootstrap.html** - Bootstrap導入後
同じデザインをBootstrapで実装。コード量の違いを比較できます。

## 💡 使い方

1. **VSCodeでファイルを開く**
2. **Live Serverで表示** - 右クリック → "Open with Live Server"
3. **ブラウザ幅を変更** - レスポンシブの動きを確認
4. **Chrome検証ツール** - F12を押してコードを確認
